# dart_object_extension_gen

[![pub](https://img.shields.io/pub/v/dart_object_extension_gen.svg?style=flat)](https://pub.dev/packages/dart_object_extension_gen)

Code generator for [dart_object_extension](https://pub.dev/packages/dart_object_extension). Generates type-safe `copyWith()` extension methods using `build_runner`.

## Setup

Add to your `pubspec.yaml`:

```yaml
dependencies:
  dart_object_extension: latest

dev_dependencies:
  build_runner: latest
  dart_object_extension_gen: latest
```

## Usage

```bash
flutter pub run build_runner build
```

For usage details and examples, see the [dart_object_extension](https://pub.dev/packages/dart_object_extension) package.
